CHW Training Simulation Platform

Practice trauma-informed conversations with AI-driven patients. Track performance, adjust emotional states, and run multi-user sessions.

How to Use

  1. Select a CHW user and a patient scenario (or load a custom patient).
  2. Adjust emotion sliders (optional).
  3. Click Push to Talk and ask open-ended, empathetic questions.
  4. Review the conversation log and scores.
Tips:
  • “How are you feeling?” → boosts openness
  • “Why…” → can increase defensiveness
  • Use reflective listening and avoid judgment
